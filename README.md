# Simple E-Commerce Web App

This is a web application built with **HTML, CSS, and JavaScript**.

## Current Features
- **Product Search:** Search for products by name.
- **Product Cards:** Products are displayed in cards with basic details.

## Planned Features
- **Add/Remove Products from Cart:** This feature is not yet implemented but will be added in future updates.

## Technologies Used
- HTML
- CSS
- JavaScript

## Notes
- This project is currently in development. Additional features, such as cart management and product addition, will be implemented soon.
